Desafio super trunfo: desenvolvendo a lógica do jogo-nivel novato

Este é um programa em C que permite comparar duas cartas de cidades com base em atributos numéricos, O usuário insere os dados das cartas e o programa compara cada atributo, indicando qual carta venceu em cada um.

Funcionalidades:

Inserção de dados para Carta 1 e Carta 2:Nome do estado, Código da carta, Nome da cidade, etc...
Comparação dos atributos entre as cartas.
Indicação de qual carta venceu em cada atributo.
Atributo Densidade Populacional vence quem tem menor valor.
Atributo Super Poder vence quem tem maior valor.

Como compilar:
No terminal, dentro da pasta onde está o arquivo .c: gcc nome_do_arquivo.c -o jogo_cartas

Para rodar:
./jogo_cartas    # Linux ou mac
jogo_cartas.exe  # Windows
